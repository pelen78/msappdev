Goal 5 — Documentation

Build your
About section

Every real app has an About page. Here's exactly what yours needs to include.

✅ About Section Checklist

0 / 6
🎉 About section complete — your app is documented like a pro!
Item 1

App Name & Description

Include your app's name and a short paragraph (2–4 sentences) explaining what it does and who it's for. Avoid technical jargon — write as if explaining it to a friend.

Example

WeatherNow — "WeatherNow gives you a clean, fast forecast for any city in the world. Just type a location and get the current temperature, humidity, and a 3-day outlook — no account needed."

Item 2

Screenshot or Logo

Show at least one visual — your app logo, an icon, or a screenshot of the main screen. This gives the About section personality and helps users recognize your app at a glance.

💡 If you have a logo, use it here. If not, a clean screenshot of your home screen works perfectly. Size it so it doesn't overwhelm the text.
Item 3

Technologies & Tools Used

List the languages, libraries, and APIs your app is built with. You can display these as tags, a simple list, or inline text. Be specific — don't just write "JavaScript", mention what you used it for.

Example

Built with HTML5 · CSS3 · JavaScript · OpenWeatherMap API

Item 4

Developer Name

Add your name (and your teammates' names if it's a group project). This is your work — own it. You can also add a short bio or your grade/class.

Example

Made by Sofia Martínez — 10th Grade, Applied Computing · ATS 2025

Item 5

Link to GitHub Repository

Add a visible link to your GitHub repo. This lets anyone browse your source code, see your commit history, and verify your work. Use a button or styled anchor — don't just paste a raw URL.

💡 Label it clearly: "View Source on GitHub" or "GitHub Repo". Open it in a new tab using target="_blank".
Item 6

AI Tools Credits

Be transparent about which AI tools you used and exactly how. This is a professional practice — real developers do this. Don't just write "used AI" — be specific about what task it helped with.

Example

🤖 ChatGPT — generated initial logo concepts and color palette ideas.
🤖 Claude — helped debug the fetch() function and suggested the card layout.
🤖 Canva AI — used to refine the final logo icon.

⚠️ If you used AI to write code, you must be able to explain every line. Never submit code you don't understand.
